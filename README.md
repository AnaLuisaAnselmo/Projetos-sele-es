# Projetos-sele-es
Sistema Copa do Mundo

Sistema desenvolvido em PHP utilizando MVC e Programação Orientada a Objetos.

## Funcionalidades
- Cadastro de grupos
- Cadastro de seleções
- Cadastro de usuários
- Cadastro de jogos
- Registro de resultados
- Classificação automática por grupo

## Tecnologias
- PHP OO
- MySQL
- HTML
- CSS
- MVC
